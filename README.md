# ShopEZ - E-Commerce Web Application

## Project Overview

ShopEZ is a Full Stack MERN E-Commerce Application developed using MongoDB, Express.js, React.js and Node.js.

The application allows customers to:

* Register and Login
* Browse Products
* Filter Products by Category
* Add Products to Cart
* Place Orders
* Track Orders

The application also provides an Admin Dashboard for:

* Product Management
* Category Management
* Banner Management
* Order Management

## Installation

### Frontend

cd client

npm install

npm start

### Backend

cd server

npm install

npm start

---

## Environment Variables

Create .env file inside server folder

PORT=6001

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key
